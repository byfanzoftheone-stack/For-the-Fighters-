# For-the-Fighters-

Migrated legacy Netlify static build with:
- Vercel-ready static hosting (`legacy_build/` via `vercel.json`)
- Railway-ready AI backend (`railway-api/`) using Anthropic API
- Vercel serverless API proxy (`api/[...path].js`) driven by `RAILWAY_API_BASE_URL`
